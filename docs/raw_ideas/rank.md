RANK

|> RARIDADE
Os rank podem ser aplicados a titulos e itens. Habilidades podem sempre ser aprendidas e personagens podem ser criados ou respawnados.Os
ranks podem ser distribuidos como:
	Comum < Incomum < Raro < Lendário < Mítico
Esses são as 5 raridades da terra, existem as 5 raridades do céu
	Transcendental < Imaterial < Caótico < Primordial < Atemporal
Essas são as raridades padrões que podem existir porém também existem as raridades especiais, que são designadas para itens com um própósito
diferente de qualquer outro item:
	Únicos : itens de eventos que só pode existir aquele e não pode ser duplicado.
	Divino : todo item que tem como origem uma divindade, pode existir réplicas porém deve existir um propósito para cada réplica.
	Demoniaco : todo item que tem como origem O Corrupto, um ser das trevas que corrompe tudo, é feito de pura trevas o item, mesma regra, precisa ter um propósito.
Itens com raridades especiais só são obtidos em condições muito especificas onde o personagem precisa estar em um evento especial ou ter itens e quests concluidas especificas.

|> DIFICULDADE
As dificuldades vão desde coleta de recursos até enfrentar um problema que é inimaginavel de que pudesse sequer ser pensado de existir.
	Facil < Medio < Dificil < Desafiador < Insano < Catastrofe < Apocaliptico

|> IMPACTO
O alcance do impacto vai desde ao ponto influenciar apenas a vida de um personagem ou uma familia até ultrapassar barreiras interdimensionais.
	Pessoal < Local < Regional < Nacional < Internacional < Global < Mundial < Interdimensional

|> PROEFICIENCIA
Esse tipo de rank pode ser aplicado tanto em itens, personagense habilidade, representando o
potencial daquilo que estiver sendo rankeado. Esse rankeamento é feito utilizando elementos da natureza
como referencia da eficiencia.
