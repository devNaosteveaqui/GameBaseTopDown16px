CHARACTER CONTROL

w : andar para frente
s : andar para tras
a : andar para esquerda
d : andar para direita
z : agachar
x : andar de costas
Space : Salto
Botão esquerdo do mouse (BEM): atacar
Botão direito do mouse (BDM): colocar
Botão central do mouse (BCM): fazer em si mesmo
i : interagir
o : abrir interface do bloco
e : abrir iventario
CapsLock : callSkill

Com o botão esquerdo do mouse pressionado o personagem tem 0.6 segundos para realizar um desses movimentos:
w/a/s/d = Manejo DOWN
ww/aa/ss/dd = Estocada
ws/ad = Manejo Aberto Lateral
wa/aw/as/sa/sd/ds/dw/wd = Manejo Fechado Lateral
Agachar = Manejo UP
Andar de costas = Defender

Salto antes do click = Manejo UP HEAVY
Salto depois do click = Manejo DOWN HEAVY
